# Smart Laundry Hub

Aplikasi Smart Laundry Hub adalah aplikasi mobile berbasis Flutter untuk mengelola layanan penatu (laundry) dengan lebih mudah, cepat, dan praktis. 

## Fitur Utama

- **Home Dashboard**: Ringkasan cucian aktif, total pengeluaran, dan status pesanan terakhir.
- **Tracking Cucian**: Pantau status pesanan (Dicuci, Dijemur, Siap Ambil, Selesai) secara real-time.
- **Kalkulator & Order**: Hitung estimasi harga berdasarkan berat (kiloan) atau jumlah (satuan) dan buat pesanan baru.
- **Riwayat Transaksi**: Lihat daftar riwayat pesanan yang sudah selesai beserta total biaya.
- **Maps**: Temukan lokasi cabang Smart Laundry terdekat.

## Getting Started

Proyek ini dibangun menggunakan [Flutter](https://flutter.dev/).

### Cara Menjalankan Aplikasi

1. Pastikan Anda telah menginstal Flutter SDK.
2. Clone repository ini.
3. Buka terminal dan arahkan ke direktori proyek.
4. Jalankan perintah `flutter pub get` untuk mengunduh semua dependensi.
5. Jalankan aplikasi di emulator atau perangkat fisik menggunakan perintah `flutter run`.
